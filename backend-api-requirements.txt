Yêu cầu API backend cho hệ thống Đông Y Pharmacy (cập nhật hoàn chỉnh):

1. API Quản lý người dùng:
- POST /api/auth/register - Đăng ký tài khoản
- POST /api/auth/login - Đăng nhập
- POST /api/auth/forgot-password - Quên mật khẩu
- POST /api/auth/reset-password - Đặt lại mật khẩu
- POST /api/auth/social-login - Đăng nhập qua mạng xã hội
- GET /api/users/me - Lấy thông tin user hiện tại
- PUT /api/users/me - Cập nhật thông tin user
- GET /api/users - Danh sách user (admin)
- GET /api/users/{id} - Chi tiết user (admin)
- PUT /api/users/{id} - Cập nhật user (admin)
- DELETE /api/users/{id} - Xóa user (admin)

2. API Quản lý địa chỉ giao hàng:
- GET /api/addresses - Lấy danh sách địa chỉ của user
- POST /api/addresses - Thêm địa chỉ mới
- PUT /api/addresses/{id} - Cập nhật địa chỉ
- DELETE /api/addresses/{id} - Xóa địa chỉ
- PUT /api/addresses/{id}/default - Đặt làm địa chỉ mặc định

3. API Wishlist (sản phẩm yêu thích):
- GET /api/wishlist - Lấy danh sách sản phẩm yêu thích
- POST /api/wishlist - Thêm sản phẩm vào wishlist
- DELETE /api/wishlist/{productId} - Xóa sản phẩm khỏi wishlist

4. API Quản lý sản phẩm:
- GET /api/products - Lấy danh sách sản phẩm (phân trang, lọc, sắp xếp)
- GET /api/products/featured - Lấy sản phẩm nổi bật
- GET /api/products/{id} - Lấy chi tiết sản phẩm
- POST /api/products - Tạo sản phẩm (admin)
- PUT /api/products/{id} - Cập nhật sản phẩm (admin)
- DELETE /api/products/{id} - Xóa sản phẩm (admin)

5. API Quản lý danh mục:
- GET /api/categories - Lấy tất cả danh mục
- GET /api/categories/{id} - Lấy sản phẩm theo danh mục
- POST /api/categories - Tạo danh mục (admin)
- PUT /api/categories/{id} - Cập nhật danh mục (admin)
- DELETE /api/categories/{id} - Xóa danh mục (admin)

6. API Giỏ hàng:
- GET /api/cart - Lấy giỏ hàng của user
- POST /api/cart - Thêm sản phẩm vào giỏ hàng
- PUT /api/cart/{itemId} - Cập nhật số lượng
- DELETE /api/cart/{itemId} - Xóa sản phẩm khỏi giỏ hàng

7. API Đơn hàng:
- GET /api/orders - Lấy danh sách đơn hàng của user
- GET /api/orders/{id} - Lấy chi tiết đơn hàng
- POST /api/orders - Tạo đơn hàng
- PUT /api/orders/{id}/cancel - Hủy đơn hàng
- GET /api/admin/orders - Danh sách tất cả đơn hàng (admin)
- PUT /api/admin/orders/{id}/status - Cập nhật trạng thái đơn hàng (admin)

8. API Thanh toán:
- POST /api/payments - Tạo thanh toán
- GET /api/payments/{id} - Lấy trạng thái thanh toán

9. API Đánh giá sản phẩm:
- GET /api/products/{id}/reviews - Lấy đánh giá sản phẩm
- POST /api/products/{id}/reviews - Tạo đánh giá
- PUT /api/reviews/{id} - Cập nhật đánh giá
- DELETE /api/reviews/{id} - Xóa đánh giá

10. API Bài viết/blog:
- GET /api/posts - Lấy danh sách bài viết
- GET /api/posts/{slug} - Lấy chi tiết bài viết
- POST /api/posts - Tạo bài viết (admin)
- PUT /api/posts/{id} - Cập nhật bài viết (admin)
- DELETE /api/posts/{id} - Xóa bài viết (admin)

11. API Liên hệ:
- POST /api/contact - Gửi liên hệ
- GET /api/contact - Lấy danh sách liên hệ (admin)
- PUT /api/contact/{id}/read - Đánh dấu đã đọc (admin)

12. API Khuyến mãi:
- GET /api/promotions - Lấy danh sách khuyến mãi
- GET /api/promotions/{code} - Kiểm tra khuyến mãi
- POST /api/promotions - Tạo khuyến mãi (admin)
- DELETE /api/promotions/{id} - Xóa khuyến mãi (admin)

13. API Đăng ký nhận tin:
- POST /api/subscriptions - Đăng ký nhận tin
- GET /api/subscriptions - Danh sách email đăng ký (admin)
- DELETE /api/subscriptions/{id} - Xóa email đăng ký (admin)
